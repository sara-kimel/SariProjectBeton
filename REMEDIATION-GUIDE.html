מדריך הרצה — תיקוני Project Sari

איך להפעיל אותי (Claude) על כל אחד משלבי התיקון שהוכנו. כל מה שצריך זה להעתיק שורה אחת ולהדביק בצ'אט חדש.

הרקע בקצרה: נעשתה סקירת קוד מלאה, 6 התיקונים הקריטיים (P0) כבר בוצעו ואומתו (52 בדיקות ירוקות), ונותרו 8 שלבים (R1–R8) שמחכים לביצוע. כל שלב מתועד כפרומפט עצמאי בתיקייה prompts/remediation/.

איך זה עובד — 3 צעדים

  1. פותחים צ'אט חדש של Claude Code בתוך תיקיית הפרויקט (Project Sari).
  2. מעתיקים את שורת ההפעלה של השלב הרצוי (למטה) ומדביקים בצ'אט. זהו — לא צריך לכתוב שום דבר נוסף.
  3. אני אקרא את קובץ הפרומפט + את ה-SPEC, אבצע את כל המשימות, אריץ את הבדיקות, ואעדכן את התיעוד. בסוף אקבל ממך אישור להמשיך לשלב הבא.

שורות ההפעלה — לפי שלב

העתיקי את השורה של השלב שאת רוצה. אפשר לבצע אחד בכל פעם (מומלץ) — כל שלב עצמאי.

R1 — ניקוי קוד מת + טיפול-שגיאות + שכבתיות P1 הקשחה

מוחק 5 קבצי קוד מת (כולל מוקש הכפלת מחיר ×2), מאחד הודעות שגיאה, מחזיר סדר שכבתי.

העתיקי והדביקי בצ'אט חדש:בצע את השלב לפי prompts/remediation/R1-cleanup-and-errors.md

R2 — הקשחת בסיס הנתונים P1 הקשחה

אינדקסים, אילוצי CHECK על סטטוס, NOT NULL, מיגרציות אידמפוטנטיות.

העתיקי והדביקי בצ'אט חדש:בצע את השלב לפי prompts/remediation/R2-db-hardening.md

R3 — הקשחת אבטחה P1 הקשחה

ביטול טוקנים בשינוי סיסמה, הקשחת rate-limit, סיסמת admin ממשתנה סביבה.

העתיקי והדביקי בצ'אט חדש:בצע את השלב לפי prompts/remediation/R3-security-hardening.md

R4 — הקשחת צד לקוח P1 הקשחה

מצבי טעינה/ריק/שגיאה בלוחות הבקרה, נגישות (aria-live), ניווט חכם בפקיעת סשן.

העתיקי והדביקי בצ'אט חדש:בצע את השלב לפי prompts/remediation/R4-client-hardening.md

R5 — מסכים חסרים והסכמת תנאים P2 השלמה

עמוד פרופיל/שינוי-סיסמה, מסכי היסטוריית עסקאות, הסכמת תנאי-שימוש בהרשמה. (חלופת המפה בוטלה — המפה עובדת.)

העתיקי והדביקי בצ'אט חדש:בצע את השלב לפי prompts/remediation/R5-screens-and-consent.md

R6 — השלמת בדיקות P2 השלמה

בדיקות לשלב-הניקוד של המנוע (כרגע 0 כיסוי!), גבולות כמות, בדיקות שליליות, בדיקות לקוח.

העתיקי והדביקי בצ'אט חדש:בצע את השלב לפי prompts/remediation/R6-tests.md

R7 — תשתית ותפעול P2 השלמה

ErrorBoundary, README אנושי, CI (GitHub Actions), Dockerfile.

העתיקי והדביקי בצ'אט חדש:בצע את השלב לפי prompts/remediation/R7-infra.md

R8 — דירוג, מוניטין ודיווח מומלץ / עתידי

דירוג הדדי אחרי עסקה, דיווח וחסימת משתמשים. אפשר לדחות בבטחה.

העתיקי והדביקי בצ'אט חדש:בצע את השלב לפי prompts/remediation/R8-ratings-reputation.md
